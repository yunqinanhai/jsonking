# jsonking
# 产品高级配置JSON校验工具

一款专为产品高级配置设计的 JSON 格式校验工具，支持语法错误检测、行号定位、字段说明搜索和示例展示。

## 功能概览

### 🔍 JSON 格式校验

- **语法检测**：自动检测括号不匹配、多余逗号、引号缺失等常见语法错误
- **行号定位**：精确提示错误所在行号及错误内容
- **实时统计**：显示总行数、字符数、字段数
- **错误高亮**：格式错误时编辑器边框变红，直观提示

### 📖 字段说明搜索

- 收录 **90+ 个产品高级配置字段**
- 支持按字段名、中文说明搜索
- 每个字段显示类型标签（string / number / boolean / array / object）
- 部分字段提供**可折叠示例**，点击即可查看完整配置写法

### 🎨 其他特性

- **深色/浅色主题**切换
- **格式化 / 压缩** JSON 内容
- **快捷模板**插入（对象、数组、嵌套）
- **快捷键**支持
- **中英双语**提示信息

## 使用方式

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖。

```bash
# 方式一：直接双击打开
index.html

# 方式二：本地服务器
python -m http.server 8086
# 访问 http://localhost:8086/index.html
```

| 区域 | 说明 |
|------|------|

| 区域 | 说明 |
|------|------|

## 校验规则

工具会对以下 JSON 语法问题进行检测：

| 检测项 | 说明 |
|--------|------|
| 括号匹配 | 检测 `{}` `[]` 是否成对出现 |
| 引号匹配 | 检测双引号是否成对出现 |
| 尾部逗号 | 检测对象/数组最后一个元素后是否有多余逗号 |
| 键名引号 | 检测键名是否用双引号包裹 |
| 字符串值引号 | 检测字符串值是否用双引号包裹 |
| 空值字段 | **允许**字段值为空字符串 `""` |

## 字段示例展示

对于结构复杂的字段（如 array、object 类型），点击「查看示例」可展开完整的配置写法，包含字段名和值，方便直接复制使用。

示例：

```
"followUpAllowedSkillList": [
    "2019030500000001",
    "2019030500000002"
]
```

```
"baConfig": {
    "sysCondition":"priority",
    "noMonitorSkillList":["skillId1", "skillId2"],
    "singleTurn":"off",
    "targetManagement":"off",
    "recommend":"off",
    "rhetoricalQuestion":"off"
}
```
